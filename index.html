Политика конфиденциальности

1. Определения

1.1. Данные аккаунта — информация, необходимая для работы игрового аккаунта.

1.2. Обработка данных — любые действия с данными.

1.3. Пользователь — лицо, использующее Приложение.

1.4. Разработчик — самозанятый разработчик Volkodavovich.

2. Какие данные обрабатываются

  • Уникальный логин (никнейм).
  • Хешированный пароль (SHA‑256).
  • Игровые показатели.
  • Дата регистрации и последнего входа.

Не обрабатываются: ФИО, email, телефон, паспорт, геолокация, контакты.

3. Хранение данных

3.1. Данные хранятся в РФ (VK Cloud).

3.2. Пароли хранятся только в виде хешей.

3.3. Данные удаляются в течение 30 дней после удаления аккаунта.

3.4. Данные не передаются за пределы РФ.

4. Передача данных

Передача возможна только по закону.

5. Статистика и аналитика

Собираются только анонимные технические данные для диагностики ошибок.

6. Права Пользователя

7. Безопасность данных

Используются технические меры защиты, включая HTTPS и хеширование.
