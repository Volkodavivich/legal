Политика конфиденциальности

1. Определения

Данные аккаунта, обработка данных, Пользователь, Разработчик.

2. Какие данные обрабатываются

2.1. Обрабатываемые данные:

  • Игровой логин.
  • Хешированный пароль (bcrypt).
  • Игровой прогресс и статистика.
  • Служебные метаданные.

2.2. Данные, которые НЕ собираются:

  • ФИО, дата рождения.
  • Контакты (кроме обращений в поддержку).
  • Документы, финданные.
  • IP, геолокация, данные устройства.
  • Crash‑отчёты, биометрия.

3. Хранение и защита данных

Данные хранятся в VK Cloud Solutions на территории РФ. Передача — по защищённым каналам.

Удаление данных:

  • Автоматически — через интерфейс.
  • По запросу — в течение 7 рабочих дней.

4. Передача данных третьим лицам

Передача возможна только по закону. Хостинг — VK Cloud Solutions.

5. Права Пользователя

  • Запросить данные.
  • Удалить данные.
  • Обращаться по email.

6. Заключительные положения

Разработчик применяет технические меры защиты. Пользователь обязан защищать своё устройство.
