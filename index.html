1. Общие положения

1.1. Приложение «Время Пришло» предназначено для развития личной продуктивности, формирования привычек и повышения мотивации в игровом формате.

1.2. Использование Приложения возможно только после регистрации и согласия Пользователя с условиями настоящего Соглашения.

1.3. Регистрация и дальнейшее использование Приложения подтверждают, что Пользователь ознакомлен с Соглашением, понимает его и принимает условия.

1.6. Разработчик вправе вносить изменения в Соглашение в порядке, предусмотренном разделом 7 настоящего Соглашения.

2. Возрастные ограничения

2.1. Приложение предназначено для лиц старше 18 лет.

Политика конфиденциальности

1. Определения

1.1. Данные аккаунта – информация, необходимая для создания и функционирования игрового аккаунта Пользователя в Приложении.

1.2. Обработка данных – любое действие (операция) или совокупность действий, совершаемых с использованием средств автоматизации или без них, включая сбор, запись, хранение, использование, передачу.

1.3. Пользователь – физическое лицо, прошедшее регистрацию и использующее Приложение.

1.4. Разработчик (Оператор) – самозанятый разработчик, действующий под псевдонимом Volkodavovich, осуществляющий обработку данных.

2. Какие данные обрабатываются и для каких целей

2.1. Приложение обрабатывает только данные, необходимые для работы игрового аккаунта:

  • Уникальный игровой логин (никнейм), выбранный Пользователем.
  • Хешированный пароль (алгоритм SHA-256) исключительно для целей аутентификации и безопасности аккаунта.
  • Игровые показатели: очки, уровень, энергия, прогресс в привычках, внутриигровая валюта и др.
  • Дата и время регистрации, последнего входа.

2.2. Приложение НЕ запрашивает и НЕ обрабатывает:

  • Имя, фамилию, отчество.
  • Адрес электронной почты (кроме контактной, если пользователь пишет в поддержку).
  • Номер телефона.
  • Паспортные данные, геолокацию, контакты из адресной книги.
  • Платёжную информацию (в бесплатной версии).

2.3. Цели обработки: предоставление функционала Приложения, идентификация пользователя в системе, сохранение игрового прогресса, обеспечение безопасности аккаунта, улучшение работы Приложения на основе анонимной статистики.

3. Хранение данных

3.1. Данные аккаунта хранятся на защищенных серверах на территории Российской Федерации (VK Cloud Solutions) в соответствии с требованиями законодательства РФ.

3.2. Пароли хранятся только в виде криптографических хешей (SHA-256). Разработчик не имеет доступа к паролям Пользователей в исходном виде.

3.3. Срок хранения данных: в течение всего периода существования аккаунта Пользователя. После удаления аккаунта данные безвозвратно удаляются в течение 30 (тридцати) дней.

3.4. Данные не передаются и не обрабатываются за пределами Российской Федерации.

4. Передача данных третьим лицам

4.1. Разработчик не передает данные аккаунта Пользователя третьим лицам, за исключением случаев, прямо предусмотренных законодательством Российской Федерации (по запросу уполномоченных государственных органов).

4.2. Для технической инфраструктуры используются услуги доверенных хостинг-провайдеров (VK Cloud). С ними заключены необходимые соглашения, обязывающие их обеспечивать конфиденциальность и безопасность данных.

5. Статистика и аналитика

5.1. Для анализа ошибок и улучшения производительности Приложение может собирать анонимные агрегированные данные: тип устройства, версия ОС, частота возникновения сбоев. Эти данные не позволяют идентифицировать личность Пользователя.

Пользователь имеет право:

6.1. На доступ к своим данным: запросить подтверждение факта обработки и список обрабатываемых данных.

6.2. На удаление данных: в любой момент удалить свой аккаунт и все связанные с ним данные через интерфейс Приложения или обратившись в поддержку.

7.2. К ним относятся использование хеширования паролей, защищенные соединения (HTTPS), регулярное обновление программного обеспечения.

7.3. ВАЖНО: Несмотря на все прилагаемые усилия, не существует способов передачи и хранения данных в интернете, которые были бы на 100% безопасны. Пользователь также обязан принимать меры для защиты своего устройства и данных для входа.
